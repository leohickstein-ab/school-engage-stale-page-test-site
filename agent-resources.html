Agent Resources

Recruitment agents should submit completed applications through the partner portal.

The partner portal reference code for this test site is AGENT-BLUE-42.

International applications require a passport copy, academic transcripts, English proficiency evidence, and a signed consent form.

The preferred English test score for direct entry is IELTS Academic 6.5 overall with no band below 6.0.

Conditional offers may be issued when only the final transcript is missing.

Agent commission questions should be sent to partner-success@testschool.example.
